Core Project Management:
1. GET /v2/projects - List projects
2. GET /v2/projects/{projectId} - Get project details
3. DELETE /v2/projects/{projectId} - Delete project
4. PATCH /v2/projects/{projectId}/description - Update project description
5. POST /v2/projects/{projectId}/avatar - Upload project avatar
6. DELETE /v2/projects/{projectId}/avatar - Delete project avatar
7. GET /v2/projects/{projectId}/jwt - Get project JWT

Project Memberships:
8. GET /v2/project-memberships - List all project memberships
9. GET /v2/projects/{projectId}/memberships - List project memberships
10. GET /v2/projects/{projectId}/memberships/self - Get own membership
11. GET /v2/project-memberships/{projectMembershipId} - Get membership details
12. PATCH /v2/project-memberships/{projectMembershipId} - Update membership
13. DELETE /v2/project-memberships/{projectMembershipId} - Remove membership
14. POST /v2/projects/{projectId}/leave - Leave project
