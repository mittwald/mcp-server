# mw database redis get

```
$ mw database redis get --help
Get a Redis database.

USAGE
  $ mw database redis get ID -o txt|json|yaml

ARGUMENTS
  ID  ID of the Redis database to retrieve.

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  Get a Redis database.

```
