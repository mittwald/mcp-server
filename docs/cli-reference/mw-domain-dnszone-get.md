# mw domain dnszone get

```
$ mw domain dnszone get --help
gets a specific zone

USAGE
  $ mw domain dnszone get DNSZONE-ID -o txt|json|yaml

ARGUMENTS
  DNSZONE-ID  ID or domain name of a DNS zone

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  gets a specific zone

```
