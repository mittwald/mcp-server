# mw database mysql

```
$ mw database mysql
List available MySQL character sets and collations, optionally filtered by a MySQLVersion.

USAGE
  $ mw database mysql COMMAND

TOPICS
  database mysql user  Create a new MySQL user

COMMANDS
  database mysql charsets      List available MySQL character sets and
                               collations, optionally filtered by a
                               MySQLVersion.
  database mysql create        Create a new MySQL database
  database mysql delete        Delete a MySQL database
  database mysql dump          Create a dump of a MySQL database
  database mysql get           Get a MySQLDatabase.
  database mysql import        Imports a dump of a MySQL database
  database mysql list          List MySQLDatabases belonging to a Project.
  database mysql phpmyadmin    Open phpMyAdmin for a MySQL database.
  database mysql port-forward  Forward the TCP port of a MySQL database to a
                               local port
  database mysql shell         Connect to a MySQL database via the MySQL shell
  database mysql versions      List available MySQL versions.

```
