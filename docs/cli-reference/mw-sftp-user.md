# mw sftp-user

```
$ mw sftp-user
Manage SFTP users of your projects

USAGE
  $ mw sftp-user COMMAND

COMMANDS
  sftp-user create  Create a new SFTP user
  sftp-user delete  Delete an SFTP user
  sftp-user list    List all SFTP users for a project.
  sftp-user update  Update an existing SFTP user

```
