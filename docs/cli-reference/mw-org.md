# mw org

```
$ mw org
Manage your organizations, and also any kinds of user memberships concerning these organizations.

USAGE
  $ mw org COMMAND

TOPICS
  org invite      Invite a user to an organization.
  org membership  List all memberships belonging to an organization.

COMMANDS
  org delete  Delete an organization
  org get     Get an organization profile.
  org invite  Invite a user to an organization.
  org list    Get all organizations the authenticated user has access to.

```
