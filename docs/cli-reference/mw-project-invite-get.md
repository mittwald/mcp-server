# mw project invite get

```
$ mw project invite get --help
Get a ProjectInvite.

USAGE
  $ mw project invite get INVITE-ID -o txt|json|yaml

ARGUMENTS
  INVITE-ID  ID of the ProjectInvite to be retrieved.

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  Get a ProjectInvite.

```
