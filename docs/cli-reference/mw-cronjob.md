# mw cronjob

```
$ mw cronjob
Manage cronjobs of your projects

USAGE
  $ mw cronjob COMMAND

TOPICS
  cronjob execution  Abort a running cron job execution.

COMMANDS
  cronjob create   Create a new cron job
  cronjob delete   Delete a cron job
  cronjob execute  Manually run a cron job
  cronjob get      Get details of a cron job
  cronjob list     List cron jobs belonging to a project.
  cronjob update   Update an existing cron job

```
