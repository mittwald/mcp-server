# mw app get

```
$ mw app get --help
Get details about an app installation

USAGE
  $ mw app get [INSTALLATION-ID] -o txt|json|yaml

ARGUMENTS
  INSTALLATION-ID  ID or short ID of an app installation; this argument is
                   optional if a default app installation is set in the context.

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  Get details about an app installation

```
