# mw app install

```
$ mw app install
Install apps in your projects

USAGE
  $ mw app install COMMAND

COMMANDS
  app install contao     Creates new Contao installation.
  app install joomla     Creates new Joomla! installation.
  app install matomo     Creates new Matomo installation.
  app install nextcloud  Creates new Nextcloud installation.
  app install shopware5  Creates new Shopware 5 installation.
  app install shopware6  Creates new Shopware 6 installation.
  app install typo3      Creates new TYPO3 installation.
  app install wordpress  Creates new WordPress installation.

```
