# mw conversation

```
$ mw conversation
Manage your support cases

USAGE
  $ mw conversation COMMAND

COMMANDS
  conversation categories  Get all conversation categories.
  conversation close       Close a conversation
  conversation create      Create a new conversation
  conversation list        Get all conversations the authenticated user has
                           created or has access to.
  conversation reply       Reply to a conversation
  conversation show        Show a conversation and message history

```
