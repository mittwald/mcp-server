# mw project

```
$ mw project
Manage your projects, and also any kinds of user memberships concerning these projects.

USAGE
  $ mw project COMMAND

TOPICS
  project filesystem  Interact with the filesystem of your project
  project invite      Invite users to your projects and manage their invitations
  project membership  Control who gets to work on your projects, and who doesn't

COMMANDS
  project create  Create a new project
  project delete  Delete a project
  project get     Get details of a project
  project list    List all projects that you have access to
  project ssh     Connect to a project via SSH
  project update  Update an existing project

```
