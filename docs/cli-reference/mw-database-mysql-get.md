# mw database mysql get

```
$ mw database mysql get --help
Get a MySQLDatabase.

USAGE
  $ mw database mysql get DATABASE-ID -o txt|json|yaml

ARGUMENTS
  DATABASE-ID  The ID or name of the database

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  Get a MySQLDatabase.

```
