# mw database redis

```
$ mw database redis
Create a new Redis database

USAGE
  $ mw database redis COMMAND

COMMANDS
  database redis create    Create a new Redis database
  database redis get       Get a Redis database.
  database redis list      List Redis databases belonging to a project.
  database redis shell     Connect to a Redis database via the redis-cli
  database redis versions  List available Redis versions.

```
