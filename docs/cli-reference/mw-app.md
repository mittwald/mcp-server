# mw app

```
$ mw app
Manage apps, and app installations in your projects

USAGE
  $ mw app COMMAND

TOPICS
  app create      Creates new custom Node.js installation.
  app dependency  Manage the system dependencies of your apps
  app install     Install apps in your projects

COMMANDS
  app copy                     Copy an app within a project
  app download                 Download the filesystem of an app within a
                               project to your local machine
  app get                      Get details about an app installation
  app list                     List installed apps in a project.
  app list-upgrade-candidates  List upgrade candidates for an app installation.
  app open                     Open an app installation in the browser.
  app ssh                      Connect to an app via SSH
  app uninstall                Uninstall an app
  app update                   Update properties of an app installation (use
                               'upgrade' to update the app version)
  app upgrade                  Upgrade app installation to target version
  app upload                   Upload the filesystem of an app to a project
  app versions                 List supported Apps and Versions

```
