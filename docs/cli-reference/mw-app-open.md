# mw app open

```
$ mw app open --help
Open an app installation in the browser.

USAGE
  $ mw app open [INSTALLATION-ID]

ARGUMENTS
  INSTALLATION-ID  ID or short ID of an app installation; this argument is
                   optional if a default app installation is set in the context.

DESCRIPTION
  Open an app installation in the browser.

  This command opens an app installation in the browser. For this to work, there
  needs to be at least one virtual host linked to the app installation.

```
