# mw project membership

```
$ mw project membership
Control who gets to work on your projects, and who doesn't

USAGE
  $ mw project membership COMMAND

COMMANDS
  project membership get       Get a ProjectMembership
  project membership get-own   Get the executing user's membership in a Project.
  project membership list      List all memberships for a Project.
  project membership list-own  List ProjectMemberships belonging to the
                               executing user.

```
