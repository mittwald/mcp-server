# mw app create

```
$ mw app create
Creates new custom Node.js installation.

USAGE
  $ mw app create COMMAND

COMMANDS
  app create node        Creates new custom Node.js installation.
  app create php         Creates new custom PHP installation.
  app create php-worker  Creates new PHP worker installation.
  app create python      Creates new custom python site installation.
  app create static      Creates new custom static site installation.

```
