# mw domain get

```
$ mw domain get --help
gets a specific domain

USAGE
  $ mw domain get DOMAIN-ID -o txt|json|yaml

ARGUMENTS
  DOMAIN-ID  ID or domain name of a domain

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  gets a specific domain

```
