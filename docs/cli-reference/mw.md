# mw

```
$ mw
Hand-crafted CLI for the mittwald API

VERSION
  @mittwald/cli/1.4.3 darwin-arm64 node-v18.20.4

USAGE
  $ mw [COMMAND]

TOPICS
  app           Manage apps, and app installations in your projects
  backup        Manage backups of your projects
  context       Save certain environment parameters for later use
  contributor   Commands for mStudio marketplace contributors
  conversation  Manage your support cases
  cronjob       Manage cronjobs of your projects
  database      Manage databases (like MySQL and Redis) in your projects
  ddev          Integrate your mittwald projects with DDEV
  domain        Manage domains, virtual hosts and DNS settings in your projects
  extension     Install and manage extensions in your organisations and projects
  login         Manage your client authentication
  mail          Manage mailboxes and mail addresses in your projects
  org           Manage your organizations, and also any kinds of user
                memberships concerning these organizations.
  project       Manage your projects, and also any kinds of user memberships
                concerning these projects.
  server        Manage your servers
  sftp-user     Manage SFTP users of your projects
  ssh-user      Manage SSH users of your projects
  user          Manage your own user account

COMMANDS
  autocomplete  Display autocomplete installation instructions.
  help          Display help for mw.
  update        update the mw CLI

```
