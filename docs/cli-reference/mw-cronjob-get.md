# mw cronjob get

```
$ mw cronjob get --help
Get details of a cron job

USAGE
  $ mw cronjob get CRONJOB-ID -o txt|json|yaml

ARGUMENTS
  CRONJOB-ID  ID of the cron job to be retrieved.

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  Get details of a cron job

```
