# mw org get

```
$ mw org get --help
Get an organization profile.

USAGE
  $ mw org get [ORG-ID] -o txt|json|yaml

ARGUMENTS
  ORG-ID  ID or short ID of an org; this argument is optional if a default org
          is set in the context.

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  Get an organization profile.

```
