# mw cronjob execution

```
$ mw cronjob execution
Abort a running cron job execution.

USAGE
  $ mw cronjob execution COMMAND

COMMANDS
  cronjob execution abort  Abort a running cron job execution.
  cronjob execution get    Get a cron job execution.
  cronjob execution list   List CronjobExecutions belonging to a Cronjob.
  cronjob execution logs   Get the log output of a cronjob execution.

```
