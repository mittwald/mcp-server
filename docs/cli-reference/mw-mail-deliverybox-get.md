# mw mail deliverybox get

```
$ mw mail deliverybox get --help
Get a specific delivery box

USAGE
  $ mw mail deliverybox get ID -o txt|json|yaml

ARGUMENTS
  ID  ID of the delivery box you want to retrieve

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  Get a specific delivery box

```
