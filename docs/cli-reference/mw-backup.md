# mw backup

```
$ mw backup
Manage backups of your projects

USAGE
  $ mw backup COMMAND

TOPICS
  backup schedule  Manage backup schedules of your projects

COMMANDS
  backup create    Create a new backup of a project
  backup delete    Delete a backup
  backup download  Download a backup to your local disk
  backup get       Show details of a backup.
  backup list      List Backups for a given Project.

```
