# mw mail address get

```
$ mw mail address get --help
Get a specific mail address

USAGE
  $ mw mail address get ID -o txt|json|yaml

ARGUMENTS
  ID  id of the address you want to get

FLAGS
  -o, --output=<option>  (required) [default: txt] output in a more machine
                         friendly format
                         <options: txt|json|yaml>

DESCRIPTION
  Get a specific mail address

```
