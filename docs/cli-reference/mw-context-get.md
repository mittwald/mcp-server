# mw context get

```
$ mw context get --help
Print an overview of currently set context parameters

USAGE
  $ mw context get [-o txt|json]

FLAGS
  -o, --output=<option>  [default: txt] The output format to use; use 'txt' for
                         a human readable text representation, and 'json' for a
                         machine-readable JSON representation.
                         <options: txt|json>

DESCRIPTION
  Print an overview of currently set context parameters

  The context allows you to persistently set values for common parameters, like
  --project-id or --server-id, so you don't have to specify them on every
  command.

```
