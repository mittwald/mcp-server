# Tool Reference Docs Site

This directory contains the API/tool reference site for Mittwald MCP.

It is generated and organized by tool domain, with one page per tool.

## Build and Run

```bash
cd docs/reference
npm ci
npm run dev
```

```bash
cd docs/reference
npm run build
```

Build output goes to `dist/`.
