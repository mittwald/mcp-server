## Extension Tools

Reference documentation for all extension management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_extension_install`](./install) | Install an extension in a project or organization. |
| [`mittwald_extension_list`](./list) | List all available extensions. |
| [`mittwald_extension_list_installed`](./list-installed) | List installed extensions in a project or organization. |
| [`mittwald_extension_uninstall`](./uninstall) | Remove an extension from an organization. |
