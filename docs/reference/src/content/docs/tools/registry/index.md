## Registry Tools

Reference documentation for all registry management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_registry_create`](./create) | Create a new registry in Mittwald. |
| [`mittwald_registry_delete`](./delete) | Delete a registry from Mittwald. |
| [`mittwald_registry_list`](./list) | List registries available in Mittwald. |
| [`mittwald_registry_update`](./update) | Update an existing registry in Mittwald. |
