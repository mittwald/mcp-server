## Sftp Tools

Reference documentation for all sftp management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_sftp_user_create`](./user-create) | Create a new SFTP user. |
| [`mittwald_sftp_user_delete`](./user-delete) | Delete an SFTP user. |
| [`mittwald_sftp_user_list`](./user-list) | List all SFTP users for a project. |
| [`mittwald_sftp_user_update`](./user-update) | Update an existing SFTP user. |
