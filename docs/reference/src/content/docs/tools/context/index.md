## Context Tools

Reference documentation for all context management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_context_get_session`](./get-session) | Get current user context from Redis session (session-aware, multi-tenant safe) |
| [`mittwald_context_reset_session`](./reset-session) | Reset (clear) all user context parameters in Redis session (session-aware, multi-tenant safe) |
| [`mittwald_context_set_session`](./set-session) | Set user context in Redis session (session-aware, multi-tenant safe) |
