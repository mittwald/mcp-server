## Certificate Tools

Reference documentation for all certificate management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_certificate_list`](./list) | List SSL/TLS certificates available for a domain. |
| [`mittwald_certificate_request`](./request) | Request a new SSL/TLS certificate for a domain using Let's Encrypt. |
