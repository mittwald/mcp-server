## Org Tools

Reference documentation for all org management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_org_delete`](./delete) | Delete an organization. This is a destructive operation and cannot be undone. |
| [`mittwald_org_get`](./get) | Get detailed information about a specific organization. |
| [`mittwald_org_invite`](./invite) | Invite a user to join an organization with a specified role. |
| [`mittwald_org_invite_list`](./invite-list) | List all invites for an organization. |
| [`mittwald_org_invite_list_own`](./invite-list-own) | List all organization invites for the executing user. |
| [`mittwald_org_invite_revoke`](./invite-revoke) | Revoke an invite to an organization. |
| [`mittwald_org_list`](./list) | Get all organizations the authenticated user has access to. |
| [`mittwald_org_membership_list`](./membership-list) | List all memberships belonging to an organization. |
| [`mittwald_org_membership_list_own`](./membership-list-own) | List all organization memberships for the authenticated user. |
| [`mittwald_org_membership_revoke`](./membership-revoke) | Revoke a user's membership to an organization. |
