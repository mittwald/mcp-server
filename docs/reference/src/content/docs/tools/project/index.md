## Project Tools

Reference documentation for all project management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_project_create`](./create) | Create a new project. |
| [`mittwald_project_delete`](./delete) | Delete a project. |
| [`mittwald_project_filesystem_usage`](./filesystem-usage) | Get a project directory filesystem usage. |
| [`mittwald_project_get`](./get) | Get details of a project. |
| [`mittwald_project_invite_get`](./invite-get) | Get details of a project invite. |
| [`mittwald_project_invite_list`](./invite-list) | List project invites. |
| [`mittwald_project_invite_list_own`](./invite-list-own) | List own project invites. |
| [`mittwald_project_list`](./list) | List all projects that you have access to. |
| [`mittwald_project_membership_get`](./membership-get) | Get details of a project membership. |
| [`mittwald_project_membership_get_own`](./membership-get-own) | Get details of own project membership. |
| [`mittwald_project_membership_list`](./membership-list) | List project memberships. |
| [`mittwald_project_membership_list_own`](./membership-list-own) | List own project memberships. |
| [`mittwald_project_ssh`](./ssh) | Connect to a project via SSH. (provides command for interactive terminal) |
| [`mittwald_project_update`](./update) | Update an existing project. |
