## App Tools

Reference documentation for all app management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_app_copy`](./copy) | Copy an app within a project. |
| [`mittwald_app_create_node`](./create-node) | Create a Node.js app. |
| [`mittwald_app_create_php`](./create-php) | Create a PHP app. |
| [`mittwald_app_create_php_worker`](./create-php-worker) | Create a PHP worker app. |
| [`mittwald_app_create_python`](./create-python) | Create a Python app. |
| [`mittwald_app_create_static`](./create-static) | Create a static app. |
| [`mittwald_app_dependency_list`](./dependency-list) | Get all available system software dependencies and optionally filter by app type or installation. |
| [`mittwald_app_dependency_update`](./dependency-update) | Update one or more system software dependencies for an app installation. |
| [`mittwald_app_dependency_versions`](./dependency-versions) | Fetch available versions for a specific system software dependency. |
| [`mittwald_app_download`](./download) | Download the filesystem of an app within a project to your local machine. |
| [`mittwald_app_get`](./get) | Get details about an app installation. |
| [`mittwald_app_install_contao`](./install-contao) | Install Contao application. |
| [`mittwald_app_install_joomla`](./install-joomla) | Install Joomla application. |
| [`mittwald_app_install_matomo`](./install-matomo) | Install Matomo application. |
| [`mittwald_app_install_nextcloud`](./install-nextcloud) | Install Nextcloud application. |
| [`mittwald_app_install_shopware5`](./install-shopware5) | Install Shopware 5 application. |
| [`mittwald_app_install_shopware6`](./install-shopware6) | Install Shopware 6 application. |
| [`mittwald_app_install_typo3`](./install-typo3) | Install TYPO3 application. |
| [`mittwald_app_install_wordpress`](./install-wordpress) | Install WordPress application. |
| [`mittwald_app_list`](./list) | List installed apps in a project. |
| [`mittwald_app_list_upgrade_candidates`](./list-upgrade-candidates) | List upgrade candidates for an app installation. |
| [`mittwald_app_open`](./open) | Open an app installation in the browser. |
| [`mittwald_app_ssh`](./ssh) | Connect to an app via SSH. |
| [`mittwald_app_uninstall`](./uninstall) | Uninstall an app. |
| [`mittwald_app_update`](./update) | Update properties of an app installation. (use upgrade to update the app version) |
| [`mittwald_app_upgrade`](./upgrade) | Upgrade app installation to target version. |
| [`mittwald_app_upload`](./upload) | Upload the filesystem of an app to a project. |
| [`mittwald_app_versions`](./versions) | List supported Apps and Versions. |
