## Server Tools

Reference documentation for all server management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_server_get`](./get) | Get server details.. Retrieves information about a specific server. |
| [`mittwald_server_list`](./list) | List servers for an organization or user.. Shows all servers accessible to the current user. |
