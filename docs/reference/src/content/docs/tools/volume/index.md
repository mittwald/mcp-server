## Volume Tools

Reference documentation for all volume management tools.

### Available Tools

| Tool | Description |
|------|-------------|
| [`mittwald_volume_create`](./create) | Create a new named volume inside a project stack. |
| [`mittwald_volume_delete`](./delete) | Delete a persistent volume. WARNING: This permanently removes stored data. |
| [`mittwald_volume_list`](./list) | List persistent volumes that belong to a project stack. |
